### Requisitos

* node
* yarn
* mongodb

1. Clonar el repositorio

2. Instalar dependecias server
    $ cd server
    $ yarn install
    ```
3. Instalar dependencias client
    $ cd client
    $ yarn install
    

## Run the app

1. Run mongodb localmente
    $ mongod
    
2. Run server
    $ cd server
    $ yarn start
    
3. Run client

    $ cd client
    $ yarn start
    ```
4. Browse to `http://localhost:3000/`
